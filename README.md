# Axiant Assignment: Notes App
Small API for Quick Notes app.

## Features
- Mock Authentication
- Bearer token auth
- Request validation
- Clean structure
- Async first

## Stack
- Python
- FastAPI
- SQLModel
- Uvicorn

## Setup

1. Install depedencies
```sh
uv sync
```

2. Run application
```sh
uv run uvicorn app:main:app --reload
```
